# To do list
Lista de tarefas usando **React** e **TypeScript**

## Funcionalidades

- Adiniocar Tarefa 
- Editar Tarefa
- Alterar Staus da Tarefa
- Excluir Tarefa

## Clonando Projeto

Clone com SSH

```bash
  git clone git@github.com:IagoMesquita/todo-list-react-ts.git
```
## Instalação

Instale project-tryunfo com npm

```bash
  cd todo-react-ts
  npm install 
```
    
## Iniciar

Execute a aplicação 

```bash
npm start
```
